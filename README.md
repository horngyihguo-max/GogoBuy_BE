
# GogoBuy 後端

這是一個基於 Java 21 與 Gradle 構建的 RESTful API 專案。

## 🛠 開發環境 (Prerequisites)

## 在開始之前，請確保你已安裝以下工具：

* JDK: 21 或以上版本

* Eclipse IDE: 建議版本 2025-09 或更新，並內建 Buildship Gradle Integration 套件（現代 Eclipse 通常已內建）。

## 匯入至 Eclipse 步驟：

1. 打開 Eclipse，點擊 File > Import...。

2. 選擇 Gradle > Existing Gradle Project，點擊 Next。

3. 在 Project root directory 選擇你剛剛 clone 的資料夾。

4. 點擊 Finish。Eclipse 會自動下載依賴並建構專案結構。
